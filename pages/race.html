Om projektet

Utmaningen: Göra en samling av kommande OCR (Obstacle Course Racing) lopp, så man inte behöver besöka flera hemsidor för att ta reda på loppen.

Lösningen: Skrev ett litet script i Node.js som "skrapar" datan från flera sidor och som sparar den i en json fil som jag sedan la in i applikationen. Byggde även en sökfunktion som filtrerar efter titel och stad för att slippa gå igenom hela listan om listan skulle vara flera 100 lopp t.ex.

Resultatet: Istället för skriva ner alla kommande lopp för hand så använder jag kraften av Node.js att göra det grova jobbet.

Teknologier

  • Node.js
  • Web Scraping
  • Javascript ES6
  • SCSS
  • HTML5
