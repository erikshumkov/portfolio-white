Om projektet

Utmaningen: Bygga en hemsida som visar bostäder som är till salu.

Lösningen: Få inspiration från andra sidor som t.ex Hemnet, och göra en egen variant. Byggd med React, React Router, Leaflet.js, HTML och SCSS.

Resultatet: En hemsida med sök och filter funktionalitet. Man kan gå in på varje objekt för mer information. Det finns även en karta var bostaden är belägen.

Teknologier

  • HTML5
  • CSS3 preprocessed with SASS
  • React
  • Javascript ES6
  • Leaflet.js
  • SPA
