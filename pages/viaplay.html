Om Projektet

Utmaningen: Göra en kopia av Viaplays landnings sida som är responsivt anpassad för mobil, iPad och större skärmar.

Lösningen: Genom att titta på Viaplays sida och sedan använda mina verktyg inom HTML och CSS för att göra en sida så liknande som möjligt. Använde mig av CSS grid och flexbox för att lägga upp allt snyggt på sidan.

Resultatet: En sida som är lik originalet. Visar mina färdigheter inom HTML och CSS. Visar också att jag kan applicera samma verktyg på andra sidor.
