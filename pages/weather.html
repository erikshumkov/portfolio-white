Weather Website

Visa hemsidan visa github repo

Utmaningen: Bygga en väder applikation med React.

Lösningen: Hämtar väder-datan från Dark Sky API. Sökfunktionen använder "Google platser" så man kan söka efter önskad stad för att länka ihop väder-datan med staden.

Resultatet: Appen söker på städer och sparar senaste sökningen i local storage.
